# Agent Instructions

- This repo is an open-source local-first notes app for rendering and editing text-like files across local repositories.
- Prefer Bun over npm.
- Use `mise` for tool version alignment.
- Keep filesystem access scoped to a user-selected workspace root.
- Never print or commit private file contents, secrets, tokens, auth files, or personal data discovered while scanning repositories.
- Update `docs/PROGRESS.md` when starting, changing direction, blocking, validating, or handing off significant work.
